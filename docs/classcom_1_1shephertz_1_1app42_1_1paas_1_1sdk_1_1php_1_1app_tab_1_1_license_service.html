Detailed Description

AppTab - License. This service provides traditional License engine. This can be useful to App developers who want to sell their applications on license keys and want to use a license manager on the cloud. It allows to create a license for a particular App. Once the license scheme is created. The App developer can issue license, revoke license and check for validity of the license.

When a license is issued a license key is generated and returned. Which is used for revoking and checking the validity of the license. The Bill service is used to find licenses issued to a particular user.
